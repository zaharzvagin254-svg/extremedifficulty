# Extreme Difficulty Mod — Forge 1.20.1

Усиливает всех враждебных мобов:
- Днём: x1.5 от ванилы (HP, урон, скорость)
- Ночью: x1.85 от ванилы
- Незер и Край: всегда ночные значения

## Как собрать .jar

### Шаг 1 — Установи JDK 17
Скачай с: https://adoptium.net/temurin/releases/?version=17
Выбери: Windows x64 → .msi → скачай и установи

### Шаг 2 — Собери мод
Открой папку с модом в проводнике.
Нажми правой кнопкой на пустом месте → "Открыть в терминале" (или PowerShell).
Введи команду:

```
.\gradlew.bat build
```

Первый раз скачает Forge и зависимости (~5-10 минут, нужен интернет).

### Шаг 3 — Найди .jar
После сборки файл будет в:
```
build\libs\extremedifficulty-1.20.1.jar
```

### Шаг 4 — Установи
Скопируй `extremedifficulty-1.20.1.jar` в папку:
```
%appdata%\.minecraft\mods\
```

Готово! Запусти Minecraft с Forge 1.20.1 и играй.
